# Three-lock locus coordinate design

This folder documents the genomic coordinates and architecture for the three catalytic-domain replacement donor candidates.

## Design logic

- OCH1/YGL038C retains amino acids 1-45 as the native anchor and replaces the downstream native region with (GGGGS)2 + ISC1alpha + TAA.
- MNN2/YBR015C retains amino acids 1-41 as the native anchor and replaces the downstream native region with (GGGGS)2 + ISC1beta + TAA.
- MNN5/YJL186W retains amino acids 1-45 as the native anchor and replaces the downstream native region with (GGGGS)2 + ISC1gamma + TAA.

BY4741 locus confirmation is included as a separate compatibility check because the donor package was designed from R64-1-1/S288C coordinate logic and then checked against the BY4741 NCBI dataset.
