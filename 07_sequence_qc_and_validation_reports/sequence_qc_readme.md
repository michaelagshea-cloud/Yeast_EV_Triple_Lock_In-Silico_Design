# Sequence QC and validation reports

This folder stores current computational validation outputs for the donor candidates.

Included validation categories:

- chimeric ORF translation checks
- reading-frame and linker-phasing checks
- terminal TAA checks
- restriction-site screening summaries
- repeat/pairwise identity summaries
- junction PCR and Sanger validation planning tables
- primer-design window tables

These files support computational validation only; they do not replace experimental sequencing.
