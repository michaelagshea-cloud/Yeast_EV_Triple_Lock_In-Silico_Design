# Full donor cassette assembly

This folder contains the three full computational HDR donor candidates.

Each donor is 2299 bp:

```text
500 bp 5-prime homology arm, including retained native anchor at its end
+ 30 bp (GGGGS)2 linker
+ 1269 bp ISC1 payload block including TAA
+ 500 bp 3-prime homology arm
```

The retained native anchor is already inside the 500 bp 5-prime homology arm. It should not be counted as an additional separate block.
